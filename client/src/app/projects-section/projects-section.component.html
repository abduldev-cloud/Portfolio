About This Project

TaskFlow is a comprehensive task management platform I developed to help teams organize, track, and complete their work efficiently. Built with Angular and Node.js, it provides a modern, intuitive interface for managing projects and tasks.

The application features a powerful Kanban board, real-time collaboration tools, advanced filtering and search capabilities, and detailed analytics dashboards. Teams can create projects, assign tasks, set deadlines, track progress, and communicate seamlessly within the platform.

Key innovations include drag-and-drop task management, customizable workflows, time tracking integration, and automated notifications to keep everyone aligned and productive.

Technical Architecture

Frontend Architecture

  • Component-based Design: Modular Angular components for boards, tasks, and dashboards
  • State Management: NgRx for centralized state management and real-time updates
  • Drag & Drop: Angular CDK for smooth drag-and-drop functionality
  • Responsive Design: Bootstrap and custom CSS for mobile-first layouts
  • TypeScript: Strong typing for better code quality and maintainability
  • Routing: Angular Router with lazy loading for optimal performance

Backend Architecture

  • RESTful API: Express.js with organized MVC route structure
  • Authentication: JWT-based secure authentication with refresh tokens
  • Real-time Updates: Socket.IO for live task updates across users
  • Middleware: Request validation, error handling, and authentication checks
  • Email Service: NodeMailer for automated notifications and reminders
  • File Upload: Multer for handling file attachments

Database Design

  • Users Table: User authentication, profiles, and team memberships
  • Projects Table: Project information with owner and team associations
  • Tasks Table: Task details with status, priority, and assignment tracking
  • Comments Table: Task comments with user mentions and timestamps
  • Time Logs: Time tracking entries linked to tasks and users
  • Relationships: Foreign keys and indexes for optimal query performance

About This Project

SavePlate is a social impact platform I developed to tackle food waste and hunger simultaneously. The platform connects restaurants, grocery stores, and individual donors with local food banks and shelters in real-time.

Built with React, Node.js, and MySQL, SavePlate features intelligent matching algorithms that connect donors with nearby recipients based on food type, quantity, and location. The platform has facilitated over 500 donations, saving an estimated 10,000+ meals from going to waste.

Key innovations include real-time geolocation matching, automated notifications for time-sensitive donations, and a comprehensive impact dashboard that shows donors the direct impact of their contributions.
